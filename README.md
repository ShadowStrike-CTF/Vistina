# Vistina

**Vistina** (codename: Victor) is part of the [Strategos](https://strategos.au)
Suite — professional forensic investigation platform.

Vistina produces structured criminal forensic output from evidence
processed by the Strategos Suite engine.

This package is a namespace holder. Installing `vistina` automatically
installs `vistina-victor`, the active development package.

## Install

pip install vistina-victor
